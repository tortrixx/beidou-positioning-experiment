# 1 声明

中国卫星导航系统管理办公室负责北斗卫星导航系统空间信号接口控制文件（以下简称 ICD）的编制、修订、发布和维护等工作，并保留本文件的最终解释权。

# 2 文件范畴

北斗卫星导航系统简称北斗系统，英文缩写为 BDS。北斗系统分为北斗一号、北斗二号和北斗三号三个建设发展阶段。

本文件定义了北斗卫星导航系统空间星座和用户终端之间公开服务信号 B2a 的相关内容。B2a 信号只在北斗三号中圆地球轨道（MEO）卫星和倾斜地球同步轨道（IGSO）卫星上播发，提供公开服务，地球静止轨道（GEO）卫星不播发B2a信号。

# 3 北斗系统概述

# 3.1 空间星座

北斗三号基本空间星座由 3 颗 GEO 卫星、3 颗 IGSO 卫星和 24颗 MEO 卫星组成，并视情部署在轨备份卫星。GEO 卫星轨道高度35786 千米，分别定点于东经 80 度、110.5 度和 140 度；IGSO 卫星轨道高度35786千米，轨道倾角 55度；MEO卫星轨道高度 21528千米，轨道倾角 55度。

# 3.2 坐标系统

北斗系统采用北斗坐标系（BeiDou Coordinate System，简称BDCS）。北斗坐标系的定义符合国际地球自转服务组织（IERS）规范，与2000中国大地坐标系（CGCS2000）定义一致（具有完全相同的参考椭球参数），具体定义如下：

# （1）原点、轴向及尺度定义

原点位于地球质心；

Z 轴指向 IERS 定义的参考极（IRP）方向；

X 轴为 IERS 定义的参考子午面（IRM）与通过原点且同 Z 轴正交的赤道面的交线；

Y 轴与 Z、X 轴构成右手直角坐标系；

长度单位是国际单位制（SI）米。

BDCS参考椭球的几何中心与地球质心重合，参考椭球的旋转轴与 Z 轴重合。BDCS 参考椭球定义的基本常数见表 3-1。

# 3.3 时间系统

北斗系统的时间基准为北斗时（BDT）。BDT采用国际单位制（SI）秒为基本单位连续累计，不闰秒，起始历元为 2006 年 1 月 1 日协调世界时（UTC）00 时 00 分 00 秒。BDT 通过 UTC（NTSC）与国际UTC 建立联系，BDT 与国际 UTC 的偏差保持在 50 纳秒以内（模 1秒）。BDT 与 UTC 之间的闰秒信息在导航电文中播报。

# 4.2 信号调制

# 4.2.1 调制方式

本节采用功率归一化的复包络形式来描述信号调制方式。

假设一个调制信号的复包络表达式为：

$$
s _ {X} (t) = s _ {X 1} (t) + \mathrm {j} s _ {X 2} (t) \tag {4-1}
$$

其中，j 是虚数单位； $s _ { X 1 } \left( t \right)$ 是复包络的实部，代表信号的同相分量；

$s _ { X 2 } \left( t \right)$ 是复包络的虚部，代表信号的正交分量。 $s _ { X } \left( t \right)$ 是基带信号形式，描述了调制信号在载波调制之前所真正包含的信号结构与信息内容。

该调制信号也可以表示为带通信号形式：

$$
S _ {X} (t) = \sqrt {2 P _ {X}} \left[ s _ {X 1} (t) \cos (2 \pi f _ {X} t) - s _ {X 2} (t) \sin (2 \pi f _ {X} t) \right] \tag {4-2}
$$

其中， $f _ { \chi }$ 为信号载波频率， $P _ { X }$ 为信号功率。 $S _ { X } \left( t \right)$ 完整地表达了一个经载波调制后的带通信号。

因此，基带信号 $s _ { X } \left( t \right)$ 和带通信号 $S _ { X } \left( t \right)$ 是同一个信号的两种表达方式，二者可以相互转换。

# 4.2.2 B2a 信号

B2a 信号的复包络可以表示为

$$
s _ {\mathrm {B 2 a}} (t) = s _ {\mathrm {B 2 a} \_ \text {d a t a}} (t) + \mathrm {j} s _ {\mathrm {B 2 a} \_ \text {p i l o t}} (t) \tag {4-3}
$$

其中，数据分量 $S _ { \mathrm { B 2 a \_ d a t a } } \left( t \right)$ 由导航电文数据 $D _ { \mathrm { B 2 a \_ d a t a } } \left( t \right)$ 和测距码 $C _ { \mathrm { B 2 a \_ d a t a } } \left( t \right)$ 调制产生；导频分量 $s _ { \mathrm { B 2 a } _ { \mathrm { - } \mathrm { p i l o t } } } \left( t \right)$ 仅包括测距码 $C _ { \mathrm { B 2 a \_ p i l o t } } \left( t \right)$ ；数据分量与导频分量的功率比为 1:1，二者均采用BPSK(10)调制方式。两分量的数学表达式如下：

$$
s _ {\mathrm {B 2 a \_ d a t a}} (t) = \frac {1}{\sqrt {2}} D _ {\mathrm {B 2 a \_ d a t a}} (t) \cdot C _ {\mathrm {B 2 a \_ d a t a}} (t) \tag {4-4}
$$

$$
s _ {\mathrm {B 2 a} \text {p i l o t}} (t) = \frac {1}{\sqrt {2}} C _ {\mathrm {B 2 a} \text {p i l o t}} (t) \tag {4-5}
$$

式中， $D _ { \mathrm { B 2 a \_ d a t a } } \left( t \right)$ 的数学定义式如下：

$$
D _ {\mathrm {B 2 a} \_ \text {d a t a}} (t) = \sum_ {k = - \infty} ^ {\infty} d _ {\mathrm {B 2 a} \_ \text {d a t a}} [ k ] p _ {T _ {\mathrm {B 2 a} \_ \text {d a t a}}} (t - k T _ {\mathrm {B 2 a} \_ \text {d a t a}}) \tag {4-6}
$$

其中， $d _ { \mathrm { B 2 a \_ d a t a } }$ 为 B2a 信号的导航电文数据码， $T _ { \mathrm { B 2 a \_ d a t a } }$ 为相应的数据码

片宽度。

$C _ { \mathrm { B 2 a \_ d a t a } } \left( t \right)$ 和 $C _ { \mathrm { B 2 a } \_ \mathrm { p i l o t } } \left( t \right)$ 测距码的数学表达式如下：

$$
C _ {\mathrm {B 2 a} \_ \text {d a t a}} (t) = \sum_ {n = - \infty} ^ {\infty} \sum_ {k = 0} ^ {N _ {\mathrm {B 2 a} \_ \text {d a t a}}} ^ {- 1} c _ {\mathrm {B 2 a} \_ \text {d a t a}} [ k ] p _ {T _ {\mathrm {c} \_ \mathrm {B 2 a}}} (t - (N _ {\mathrm {B 2 a} \_ \text {d a t a}} n + k) T _ {\mathrm {c} \_ \mathrm {B 2 a}}) \tag {4-7}
$$

$$
C _ {\mathrm {B} 2 \mathrm {a} _ {-} \text {p i l o t}} (t) = \sum_ {n = - \infty} ^ {\infty} \sum_ {k = 0} ^ {N _ {\mathrm {B} 2 \mathrm {a} _ {-} \text {p i l o t}}} ^ {- 1} c _ {\mathrm {B} 2 \mathrm {a} _ {-} \text {p i l o t}} [ k ] p _ {T _ {\mathrm {c} _ {-} \mathrm {B} 2 \mathrm {a}}} (t - (N _ {\mathrm {B} 2 \mathrm {a} _ {-} \text {p i l o t}} n + k) T _ {\mathrm {c} _ {-} \mathrm {B} 2 \mathrm {a}}) \tag {4-8}
$$

其中， $c _ { \mathrm { { B 2 a } \_ d a t a } }$ 和 $c _ { \mathrm { B 2 a } \_ \mathrm { p i l o t } }$ 分别为数据分量和导频分量的测距码序列（取值为 $\pm 1$ ）， $N _ { \mathrm { B 2 a \_ d a t a } }$ 和 $N _ { \mathrm { B 2 a \_ p i l o t } }$ 为对应分量的测距码码长，其值均为 10230；$T _ { \mathrm { c } \_ \mathrm { B 2 a } } { = } 1 / R _ { \mathrm { c } \_ \mathrm { B 2 a } }$ 为 B2a 信号的测距码码片宽度，Rc_B2a =10.23 Mbps 为 B2a信号的测距码速率。

# 4.4 极化方式

卫星发射信号为右旋圆极化（RHCP）。

# 4.5载波相位噪声

未调制载波的相位噪声谱密度应满足单边噪声带宽为 10Hz 的三阶锁相环的载波跟踪精度达到 0.1弧度（RMS）。

# 4.6 杂散

卫星发射的杂散信号不超过-50dBc。

# 4.7 相关损耗

由星上载荷失真导致的相关损耗不超过 0.6dB。

# 4.8数据/码一致性

每个数据符号的边沿与码片的边沿对齐，周期性的测距码第一个码片的起始时刻与数据符号位起始时刻对齐。每个子码的边沿与主码的边沿对齐，主码第一码片起始时刻与子码码片起始时刻对齐。

# 4.9信号一致性

任意信号分量间伪码相位互差不超过 10纳秒。

# 4.10地面接收功率电平

当卫星仰角大于 5度，在地球表面附近的接收机右旋圆极化天线为0dBi增益（或线性极化天线为 3 dBi增益）时，卫星发射的导航信号到达接收机天线输出端的最小功率电平见表 4-4。

# 5.2.2 B2a 信号子码

对于不同的卫星，B2a 数据分量的子码相同，B2a 导频分量的子码不同。

（1）B2a数据分量子码

B2a数据分量的子码码长为 5，采用固定的5位码序列作为子码，子码序列为 00010，高位(MSB)先传。

（2）B2a 导频分量子码

B2a 信号导频分量子码由 Weil 码通过截断产生，其生成方式描述如下：

一个码长为 N 的 Weil 码序列可定义为

$$
W (k; w) = L (k) \oplus L ((k + w) \bmod N), k = 0, 1, 2, \dots , N - 1 \tag {5-3}
$$

式中， $L ( k )$ 是码长为 N 的 legendre 序列； $w$ 表示两个 legendre 序列之间的相位差。码长为 N 的 legendre 序列 $L ( k )$ 可根据下式定义产生：

$$
L (k) = \left\{ \begin{array}{l l} 0, k = 0 \\ 1, k \neq 0, & \text {且 存 在 整 数} x, \text {使 得} k = x ^ {2} \bmod N \\ 0, & \text {其 它} \end{array} \right. \tag {5-4}
$$

其中，mod 表示模除运算。

通过对上述码长为 N 的 Weil 码序列进行循环截取，可得到码长为 $\Nu _ { 0 }$ 的测距码，即截断序列为：

$$
c (n; w; p) = W ((n + p - 1) \bmod N; w), n = 0, 1, 2, \dots , N _ {0} - 1 \tag {5-5}
$$

式中， $p$ 为截取点，表示从 Weil 码的第 $p$ 位开始截取，取值范围为1~N。

B2a 导频分量的子码码长为 100，由长度为 1021 的 Weil 码通过截断得到， $w$ 取值范围是1到510。

B2a导频分量子码的具体参数见表5-4。表中，头 24个码片和末尾 24 个码片用八进制表示，高位(MSB)先传。

B-CNAV2 导航电文采用了循环冗余校验（CRC），具体实现方式为 CRC-24Q，其生成多项式为：

$$
g (x) = \sum_ {i = 0} ^ {2 4} g _ {i} x ^ {i} \tag {6-1}
$$

其中， $\begin{array} { r } { { \boldsymbol g } _ { i } = \left\{ \begin{array} { l l } { \boldsymbol 1 , } \\ { \boldsymbol 0 , } \end{array} \right. } \end{array}$ 0,1,3,4,5,6,7,10,11,14,17,18,23,24i 其它 。

$g \left( x \right)$ 可表示为：

$$
g (x) = (1 + x) p (x) \tag {6-2}
$$

式中， $p \left( x \right) = x ^ { 2 3 } + x ^ { 1 7 } + x ^ { 1 3 } + x ^ { 1 2 } + x ^ { 1 1 } + x ^ { 9 } + x ^ { 8 } + x ^ { 7 } + x ^ { 5 } + x ^ { 3 } + 1 \mathrm { ~ o ~ }$ 。

长度为 $k$ 的信息序列 $m _ { i }$ （ $\displaystyle i = 1 \sim k$ ）可以表示为以下多项式：

$$
m (x) = m _ {k} + m _ {k - 1} x + m _ {k - 2} x ^ {2} + \dots + m _ {1} x ^ {k - 1} \tag {6-3}
$$

多项式 $m ( x ) x ^ { 2 4 }$ 除以生成多项式 $g \left( x \right)$ ，得到余式如下：

$$
R (x) = p _ {2 4} + p _ {2 3} x + p _ {2 2} x ^ {2} + \dots + p _ {1} x ^ {2 3} \tag {6-4}
$$

式中， $p _ { 1 } \ p _ { 2 } \ldots \ldots p _ { 2 4 }$ $p _ { 2 4 }$ 依次输出构成 CRC 校验序列。

实现过程中，寄存器初始值设为全 0。

# 7.5.2 用户算法

接收机用户可通过下式计算出信号发射时刻的 BDT 时间：

$$
t = t _ {\mathrm {s v}} - \Delta t _ {\mathrm {s v}} \tag {7-1}
$$

式中，t为信号发射时刻的 BDT时间，单位为秒； $t _ { \mathrm { s v } }$ 为信号发射时刻的卫星测距码相位时间，单位为秒； $\Delta t _ { \mathrm { s v } }$ 为卫星测距码相位时间偏移，单位为秒，由下式给出：

$$
\Delta t _ {\mathrm {s v}} = a _ {0} + a _ {1} \left(t - t _ {\mathrm {o c}}\right) + a _ {2} \left(t - t _ {\mathrm {o c}}\right) ^ {2} + \Delta t _ {r} \tag {7-2}
$$

式中， $t$ 可用 $t _ { \mathrm { s v } }$ 替代； $\Delta t _ { r }$ 是相对论校正项，单位为秒，其值为：

$$
\Delta t _ {r} = F \cdot e \cdot \sqrt {A} \cdot \sin E _ {k} \tag {7-3}
$$

式中， $e$ 为卫星轨道偏心率，由本卫星星历参数得到；

$\sqrt { A }$ 为卫星轨道长半轴的开方，由本卫星星历参数计算得到；

$E _ { k }$ 为卫星轨道偏近点角，由本卫星星历参数计算得到；

$$
\mathrm {F} = - 2 \mu^ {1 / 2} / \mathrm {C} ^ {2};
$$

$\mu { = } 3 . 9 8 6 0 0 4 4 1 8 { \times } 1 0 ^ { 1 4 } \mathrm { m } ^ { 3 } \big / \mathrm { s } ^ { 2 }$ 为地心引力常数；

$\mathbf { C } = 2 . 9 9 7 9 2 4 5 8 { \times } 1 0 ^ { 8 } { \mathrm { m } } / { \mathrm { s } }$ 为光速。

# 7.8.2 单频用户算法

北斗全球电离层延迟修正模型（BDGIM）以改进的球谐函数为基础，用户接收机根据 BDGIM计算电离层延迟改正值的具体公式如下：

$$
T _ {i o n} = M _ {\mathrm {F}} \cdot \frac {4 0 . 2 8 \times 1 0 ^ {1 6}}{f ^ {2}} \cdot \left[ A _ {0} + \sum_ {i = 1} ^ {9} \alpha_ {i} A _ {i} \right] \tag {7-6}
$$

式中， $T _ { i o n }$ 为卫星与接收机视线方向的电离层延迟改正值，单位为米；$M _ { \mathrm { F } }$ 为投影函数，用于垂向和斜向电离层总电子含量（TEC）之间的转换，计算公式见式(7-17)； $f$ 为当前信号对应的载波频率，单位为赫兹； $\alpha _ { i } \left( i = 1 \sim 9 \right)$ 为电离层延迟改正模型参数（见表 7-10），单位为TECu； $A _ { i } \left( i = 1 \sim 9 \right)$ 为根据式(7-11)计算得到的数值； $A _ { 0 }$ 为根据式(7-14)计算得到的电离层延迟预报值，单位为 TECu。

用户接收机采用BDGIM计算卫星与接收机视线方向电离层延迟的具体步骤如下：

# （1）电离层穿刺点位置的计算

以 $\psi$ 表示用户和电离层穿刺点之间的地心张角，单位为弧度，其计算公式为：

$$
\psi = \frac {\pi}{2} - E - \arcsin \left(\frac {\mathrm {R e}}{\mathrm {R e} + \mathrm {H} _ {\mathrm {i o n}}} \cdot \cos E\right) \tag {7-7}
$$

式中， $E$ 表示卫星高度角，单位为弧度； $\mathrm { H } _ { \mathrm { i o n } }$ 表示电离层薄层高度；Re

表示地球平均半径。

电离层穿刺点在地球表面投影的地理纬度 $\varphi _ { g }$ 和地理经度 $\lambda _ { g }$ 的计算公式为：

$$
\left\{ \begin{array}{l} \varphi_ {g} = \arcsin (\sin \varphi_ {u} \cdot \cos \psi + \cos \varphi_ {u} \cdot \sin \psi \cdot \cos A) \\ \lambda_ {g} = \lambda_ {u} + \arctan \left(\frac {\sin \psi \cdot \sin A \cdot \cos \varphi_ {u}}{\cos \psi - \sin \varphi_ {u} \cdot \sin \varphi_ {g}}\right) \end{array} \right. \tag {7-8}
$$

式中， $\varphi _ { u }$ 表示用户地理纬度， $\lambda _ { u }$ 表示用户地理经度， $A$ 表示卫星方位角，单位均为弧度。

地固坐标系下，电离层穿刺点在地球表面投影的地磁纬度 $\varphi _ { m }$ 和地磁经度 $\lambda _ { m }$ 的计算公式为：

$$
\left\{ \begin{array}{l} \varphi_ {m} = \arcsin \left(\sin \varphi_ {\mathrm {M}} \cdot \sin \varphi_ {g} + \cos \varphi_ {\mathrm {M}} \cdot \cos \varphi_ {g} \cdot \cos \left(\lambda_ {g} - \lambda_ {\mathrm {M}}\right)\right) \\ \lambda_ {m} = \arctan \left(\frac {\cos \varphi_ {g} \cdot \sin \left(\lambda_ {g} - \lambda_ {\mathrm {M}}\right) \cdot \cos \varphi_ {\mathrm {M}}}{\sin \varphi_ {\mathrm {M}} \cdot \sin \varphi_ {m} - \sin \varphi_ {g}}\right) \end{array} \right. \tag {7-9}
$$

式中， $\varphi _ { \mathrm { { M } } }$ 为地磁北极的地理纬度， $\lambda _ { \mathrm { { u } } }$ 为地磁北极的地理经度，单位均为弧度。

日固坐标系下，电离层穿刺点的地磁纬度 $\varphi ^ { \prime }$ 和地磁经度 $\lambda ^ { \prime }$ 的计算公式为：

$$
\left\{ \begin{array}{l} \varphi^ {\prime} = \varphi_ {m} \\ \lambda^ {\prime} = \lambda_ {m} - \arctan \left(\frac {\sin \left(S _ {l o n} - \lambda_ {\mathrm {M}}\right)}{\sin \varphi_ {\mathrm {M}} \cdot \cos \left(S _ {l o n} - \lambda_ {\mathrm {M}}\right)}\right) \end{array} \right. \tag {7-10}
$$

式中， $S _ { \iota o n }$ 为平太阳地理经度，单位为弧度； $S _ { l o n }$ 的计算公式为$S _ { l o n } = \pi \cdot \Big ( 1 - 2 \cdot \big ( t - \mathrm { i n t } \big ( t \big ) \big ) \Big )$ 。其中， $t$ 表示计算时刻，以约化儒略日（MJD）表示，单位为天； $\mathrm { i n t } ( \cdot )$ 表示向下取整。

$$

# 7.8.3 双频用户算法

对于使用 B1C 和 B2a 信号的双频用户，采用双频无电离层组合伪距算法来修正电离层延迟的影响，计算方法为：

（1）对于使用 B1C导频分量和B2a导频分量的双频用户

$$
P R _ {\mathrm {B 1 C p - B 2 a p}} = \frac {P R _ {\mathrm {B 2 a p}} - k _ {1 2} \cdot P R _ {\mathrm {B 1 C p}}}{1 - k _ {1 2}} - \frac {\mathrm {C} \cdot \left(\mathrm {T} _ {\mathrm {G D B 2 a p}} - k _ {1 2} \cdot \mathrm {T} _ {\mathrm {G D B 1 C p}}\right)}{1 - k _ {1 2}} \tag {7-18}
$$

（2）对于使用 B1C导频分量和B2a数据分量的双频用户

$$
P R _ {\mathrm {B 1 C p - B 2 a d}} = \frac {P R _ {\mathrm {B 2 a d}} - k _ {1 2} \cdot P R _ {\mathrm {B 1 C p}}}{1 - k _ {1 2}} - \frac {\mathrm {C} \cdot \left(\mathrm {T} _ {\mathrm {G D B 2 a p}} + \mathrm {I S C} _ {\mathrm {B 2 a d}} - k _ {1 2} \cdot \mathrm {T} _ {\mathrm {G D B 1 C p}}\right)}{1 - k _ {1 2}} \tag {7-19}
$$

（3）对于使用 B1C数据分量和B2a导频分量的双频用户

$$
P R _ {\mathrm {B 1 C d - B 2 a p}} = \frac {P R _ {\mathrm {B 2 a p}} - k _ {1 2} \cdot P R _ {\mathrm {B 1 C d}}}{1 - k _ {1 2}} - \frac {\mathrm {C} \cdot \left(\mathrm {T} _ {\mathrm {G D B 2 a p}} - k _ {1 2} \cdot \mathrm {T} _ {\mathrm {G D B 1 C p}} - k _ {1 2} \cdot \mathrm {I S C} _ {\mathrm {B 1 C d}}\right)}{1 - k _ {1 2}} \tag {7-20}
$$

（4）对于使用 B1C数据分量和B2a数据分量的双频用户

$$
P R _ {\mathrm {B} 1 \mathrm {C d} - \mathrm {B} 2 \mathrm {a d}} = \frac {P R _ {\mathrm {B} 2 \mathrm {a d}} - k _ {1 2} \cdot P R _ {\mathrm {B} 1 \mathrm {C d}}}{1 - k _ {1 2}} - \tag {7-21}
$$

$$
\frac {\mathrm {C} \cdot \left(\mathrm {T} _ {\mathrm {G D B 2 a p}} + \mathrm {I S C} _ {\mathrm {B 2 a d}} - k _ {1 2} \cdot \mathrm {T} _ {\mathrm {G D B 1 C p}} - k _ {1 2} \cdot \mathrm {I S C} _ {\mathrm {B 1 C d}}\right)}{1 - k _ {1 2}} \tag {7-21}
$$

其中，

$$
k _ {1 2} = \left(\frac {1 5 7 5 . 4 2}{1 1 7 6 . 4 5}\right) ^ {2}, \text {为 与 频 率 有 关 的 因 子};
$$

$P R _ { \mathrm { { B 1 C p - B 2 a p } } }$ 为 B1C 导频分量和 B2a 导频分量的双频无电离层组合伪距；

$P R _ { \mathrm { { B 1 C p - B 2 a d } } }$ 为 B1C 导频分量和 B2a 数据分量的双频无电离层组合伪距；

$P R _ { \mathrm { { B 1 C d - B 2 a p } } }$ 为 B1C 数据分量和 B2a 导频分量的双频无电离层组合

伪距；

$P R _ { \mathrm { { B i C d - B 2 a d } } }$ 为 B1C 数据分量和 B2a 数据分量的双频无电离层组合伪距；

$P R _ { \mathrm { { B 1 C p } } }$ 为 B1C 导频分量的观测伪距（经钟差修正但不经 TGDB1Cp修正）；

$P R _ { \mathrm { { B l C d } } }$ 为 B1C 数据分量的观测伪距（经钟差修正但不经 TGDB1Cp、ISCB1Cd 修正）；

$P R _ { \mathrm { { B 2 a p } } }$ 为 B2a 导频分量的观测伪距（经钟差修正但不经 TGDB2ap修正）；

$P R _ { \mathrm { { B 2 a d } } }$ 为 B2a 数据分量的观测伪距（经钟差修正但不经 TGDB2ap、$\mathrm { I S C } _ { \mathrm { B 2 a d } }$ 修正）；

TGDB1Cp为 B1C 导频分量时延差；

$\mathrm { T _ { G D B 2 a p } }$ 为 B2a 导频分量时延差；

$\mathrm { I S C } _ { \mathrm { B l C d } }$ 为 B1C 数据分量相对于 B1C 导频分量的时延修正项，具体定义参见《北斗卫星导航系统空间信号接口控制文件公开服务信号B1C（1.0 版）》；

# 7.9.2 用户算法

接收机用户可通过下式计算出信号发射时刻的 BDT 时间：

$$
t = t _ {\mathrm {s v}} - \Delta t _ {\mathrm {s v}} \tag {7-22}
$$

其中， $t$ 的单位为秒； $t _ { \mathrm { s v } }$ 为信号发射时刻的卫星测距码相位时间，单位为秒； $\Delta t _ { \mathrm { s v } }$ 为卫星测距码相位时间偏移，单位为秒。

$$
\Delta t _ {\mathrm {s v}} = a _ {f 0} + a _ {f 1} \left(t - t _ {\mathrm {o a}}\right) \tag {7-23}
$$

其中，历书参考时刻 $t _ { \mathrm { o a } }$ 是以历书周计数（WNa）的起始时刻为基准的。

BDT和 UTC的时间偏差计算方法分为三种情况：

$$
t _ {\mathrm {U T C}} = \left(t _ {\mathrm {E}} - \Delta t _ {\mathrm {U T C}}\right) \bmod 8 6 4 0 0 \tag {7-24}
$$

$$
\Delta t _ {\mathrm {U T C}} = \Delta t _ {\mathrm {L S}} + A _ {\mathrm {0 U T C}} + A _ {\mathrm {1 U T C}} \left(t _ {\mathrm {E}} - t _ {\mathrm {o t}} + 6 0 4 8 0 0 \left(\mathrm {W N} - \mathrm {W N} _ {\mathrm {o t}}\right)\right) + \tag {7-25}
$$

$$
A _ {2 \mathrm {U T C}} \left(t _ {\mathrm {E}} - t _ {\mathrm {o t}} + 6 0 4 8 0 0 \left(\mathrm {W N} - \mathrm {W N} _ {\mathrm {o t}}\right)\right) ^ {2}
$$

$t _ { \mathrm { E } }$ 是指用户估计的 BDT 时间。

（2）用户当前时间处于由闰秒参考周数 $\mathrm { W N } _ { \mathrm { L S F } }$ 与日计数 DN 确定的闰秒时刻的前 6 小时与闰秒后 6 小时之内。

$$
t _ {\mathrm {U T C}} = \mathrm {W} \bmod (8 6 4 0 0 + \Delta t _ {\mathrm {L S F}} - \Delta t _ {\mathrm {L S}}) \tag {7-26}
$$

$$
\mathrm {W} = \left(\left(t _ {\mathrm {E}} - \Delta t _ {\mathrm {U T C}} - 4 3 2 0 0\right) \bmod 8 6 4 0 0\right) + 4 3 2 0 0 \tag {7-27}
$$

式中， $\Delta t _ { \mathrm { U T C } }$ 的计算方法见式(7-25)。

（3）由闰秒参考周数 $\mathrm { W N } _ { \mathrm { L S F } }$ 与日计数 DN 确定的闰秒时刻已经过去，并且用户当前时间与闰秒时刻之差大于 6个小时。

$$

$$

# 附录 多进制 LDPC 编译码方法及示例

# 1. 多进制 LDPC 编码

由多进制 LDPC（n,k）编码的校验矩阵 ${ \bf H } = [ { \bf H } _ { 1 } , { \bf H } _ { 2 } ]$ 可以计算得到生成矩阵 G。再利用生成矩阵 $\mathbf { G }$ ，对长度为 $\mathrm { k \Omega }$ 的输入信息序列 $\mathbf { m }$ 进行编码，可得到长度为 $\mathbf { n }$ 的码字 $\mathbf { c } = ( \mathbf { c _ { 0 } } , \mathbf { c _ { 1 } } , . . . , \mathbf { c _ { n - 1 } } ) { \mathbf { = m } } \cdot \mathbf { G } = [ \mathbf { m } , \mathbf { p } ]$ 。其中，$\mathbf { c } _ { j } \left( 0 \leq j < n \right)$ 为码字c中的第 $j$ 个码字符号， $\mathbf { p } { = } \mathbf { m } { \cdot } ( \mathbf { H } _ { 2 } ^ { \ - 1 } { \cdot } \mathbf { H } _ { 1 } ) ^ { \mathrm { T } }$ 为校验序列。

生成矩阵 $\mathbf { G }$ 的产生方法如下：

第 1 步：将大小为 $( n { - } k ) \times n$ 的矩阵 $\mathrm { H }$ 表示为 ${ \bf H } = [ { \bf H } _ { 1 } , { \bf H } _ { 2 } ]$ ，其中 $\mathbf { H } _ { 1 }$ 大小为 $( n { - } k ) { \times } k$ ， $\mathbf { H } _ { 2 }$ 大小为 $( n { - } k ) \times ( n { - } k )$ ；

第2步：将矩阵 $\mathbf { H }$ 转化为系统形式，用 ${ \mathbf { H } _ { 2 } } ^ { - 1 }$ 左乘 $\mathbf { H }$ ，得到右边为单位阵的校验矩阵 $\overset { \wedge } { \mathbf { H } } = [ \mathbf { H } _ { 2 } ^ { - 1 } \cdot \mathbf { H } _ { 1 } , \mathbf { I } _ { n - k } ]$ ，其中 $\mathbf { I } _ { n - k }$ 为 $( n { - } k ) \times ( n { - } k )$ 单位阵；

第 3 步：得到矩阵 $\mathbf G = [ \mathbf { I } _ { k } , ( \mathbf { H } _ { 2 } ^ { - 1 } \cdot \mathbf { H } _ { 1 } ) ^ { \mathrm { T } } ]$ ，其中 $\mathbf { I } _ { \boldsymbol { k } }$ 为 $k { \times } k$ 单位阵。

# 2. 多进制 LDPC 译码

多进制 LDPC 编码生成的码字 $\mathbf { c } = ( \mathbf { c _ { 0 } } , \mathbf { c _ { 1 } } , . . . , \mathbf { c _ { n - 1 } } )$ 在信号调制后进行信道传输，经过信道传输后，接收端可得到相应的接收序列$\mathbf { y } = ( \mathbf { y _ { 0 } } , \mathbf { y _ { 1 } } , . . . , \mathbf { y _ { n - 1 } } )$ 。其中， $\mathbf { y _ { j } } = ( y _ { j , 0 } , y _ { j , 1 } , . . . , y _ { j , r - 1 } )$ 为码字符号 $\mathbf { c } _ { j }$ 对应的信道接收信息， $\mathbf { c } _ { j } \in \mathrm { G F } ( q ) , q = 2 ^ { r }$ ， $0 \leq j < n$ 。

利用多进制 LDPC码的校验矩阵 $\mathbf { H }$ 可以对接收序列 $\mathbf { y }$ 进行校验，具体方法是：对接收序列 $\mathbf { y }$ 进行逐比特硬判决，得到硬判决码字$\hat { \mathbf { c } } = ( \hat { \mathbf { c } } _ { \mathbf { 0 } } , \hat { \mathbf { c } } _ { \mathbf { 1 } } , . . . , \hat { \mathbf { c } } _ { \mathbf { n - 1 } } )$ ，据此计算校验和 $\mathbf { s } = \hat { \mathbf { c } } \mathbf { H } ^ { \mathrm { T } }$ 。若 $\mathbf { s } = \mathbf { 0 }$ ，即对任意i $, \ 0 \leq i < m$ ，均满足 ,ˆ 0j i jc h  （有限域的乘法和加法运算），则将 cˆ 作为正确的译 $\sum _ { j \in N _ { i } } \hat { c } _ { j } h _ { i , j } = 0$ 码结果输出，否则表明cˆ是错误的译码信息。

校验矩阵H给出了 LDPC 码校验节点和变量节点之间的连接关系，相互连接的校验节点和变量节点之间能够传递置信度信息。对于大小为 $m \times n$ 的校验矩阵 $\mathbf { H }$ ，记 $h _ { i , j } \in G F ( q )$ 为 $\mathbf { H }$ 矩阵中第 $i$ 行第 $j$ 列的元素， $\mathbf { H }$ 中的每一行对应一个校验节点 CN，每一列对应一个变量节点VN。两个序数集合定义如下：

$$
M _ {j} = \{i: 0 \leq i <   m, h _ {i, j} \neq 0 \}, 0 \leq j <   n
$$

$$
N _ {i} = \{j: 0 \leq j <   n, h _ {i, j} \neq 0 \}, 0 \leq i <   m
$$

若 $h _ { i , j } \neq 0$ ，则校验节点 $\mathrm { C N } _ { i }$ 与变量节点 $\mathrm { V N } _ { j }$ 之间相连并可以相互传递置信度信息。由变量节点 $\mathrm { V N } _ { j }$ 传递给相连校验节点 $\mathrm { C N } _ { i }$ （ $i \in M _ { j }$ ）的置信度向量表示为 $\mathrm { V } 2 \mathrm { C } _ { j  i }$ ，可用于计算校验节点 $\mathrm { C N } _ { i }$ 的校验和；由校验节点 $\mathrm { C N } _ { i }$ 传递给相连变量节点 $\mathrm { V N } _ { j }$ （ $j \in N _ { i }$ ）的置信度向量表示为$\mathrm { C } 2 \mathrm { V } _ { i  j }$ ，可用于估算变量节点 $\mathrm { V N } _ { j }$ 的符号值。采用置信度传播译码算法，迭代更新 $\mathrm { V } 2 \mathrm { C } _ { j  i }$ 和 $\mathrm { C } 2 \mathrm { V } _ { i  j }$ ，能够纠正接收序列y，从而实现对传输码字c的估计。

# （1）扩展最小和法

# （2）固定路径译码算法
